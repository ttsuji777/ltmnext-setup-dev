================================================
デバッグ・セッション
================================================

- BIG-IP Nextでは、インスタンスにリモートログインして、データプレーンの問題のトラブルシューティングを行うことができます。
- BIG-IP Nextはコンテナをベースとしたアーキテクチャであるため、デバッグ用のコンテナにアクセスして、コンテナ内でさまざまなコマンドを実行します。
- デバッグ・ユーティリティは、BIG-IP Next Central ManagerのGUI、もしくはNextインスタンスへのAPIコールによって、有効化および無効化できます。
- NextインスタンスへのSSHログインは、デバッグ・セッションが有効な間のみ実行可能です。
